# 可调整分隔条故障排除指南

## 问题：PC端看不到分隔条

### 症状
- 在PC端（≥1024px）访问页面时，看不到左侧侧边栏和右侧内容区之间的分隔条

### 解决方案

#### ✅ 已优化（最新版本）

分隔条现已优化为更明显的样式：

1. **宽度增加**：从1px增加到4px
2. **颜色增强**：
   - 默认：灰色渐变背景 (#4b5563 → #374151)
   - 悬停：青蓝色发光条 (#06b6d4)
   - 拖拽：青蓝色渐变 + 光晕效果
3. **始终可见**：添加了细微的垂直细线提供基本视觉提示
4. **交互反馈**：
   - 鼠标悬停：显示30px高的青蓝色发光条
   - 拖拽时：整个区域高亮并添加阴影效果

### 如何验证

1. **确保浏览器窗口宽度 ≥ 1024px**
2. **查看侧边栏和主内容区之间**：
   - 默认应该看到一个4px宽的灰色竖条
   - 鼠标悬停时会变为青蓝色并发光
   - 按下鼠标左键可以拖拽调整宽度

### 视觉表现

| 状态 | 外观 |
|------|------|
| 默认 | 4px 灰色渐变分隔条 |
| 悬停 | 青蓝色发光条（30px高） |
| 拖拽 | 青蓝色背景 + 光晕效果 |

### 技术细节

```typescript
// 分隔条样式
style={{
  width: '4px',
  minWidth: '4px',
  maxWidth: '4px',
  background: isDragging
    ? 'linear-gradient(to bottom, #06b6d4, #0891b2)'
    : 'linear-gradient(to bottom, #4b5563, #374151)'
}}
```

### 如果仍然看不到

1. **清除浏览器缓存**：Ctrl+F5 (Windows/Linux) 或 Cmd+Shift+R (Mac)
2. **检查屏幕缩放**：确保不是缩放导致
3. **尝试不同浏览器**：Chrome / Firefox / Safari / Edge
4. **检查分辨率**：确保显示器分辨率设置正确

### 响应式说明

- **PC端 (lg: ≥1024px)**：显示分隔条，支持拖拽
- **移动端 (<1024px)**：隐藏分隔条，使用汉堡菜单切换

## 更新历史

- **v1.0**：初始版本（1px宽度，难以看见）
- **v2.0**：优化版本（4px宽度，渐变背景，悬停提示）
- **v2.1**：增强版本（添加光晕效果，始终可见的细线）
